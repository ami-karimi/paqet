# paqet
روش استفاده:
۱. در سرور (خارج):

bash <(curl -Ls https://raw.githubusercontent.com/ami-karimi/paqet/master/setup.sh)


دسترسی اجرا بدهید: chmod +x setup-paqet.sh

اجرا کنید: sudo ./setup-paqet.sh

گزینه 1 (سرور خارج) را انتخاب کنید.

پورت (مثلاً ۹۹۹۹) و پسورد را وارد کنید.

۲. در سرور (ایران):

اسکریپت را اجرا کنید.

گزینه 2 (سرور ایران) را انتخاب کنید.

همان پورت و پسورد را وارد کنید.

وقتی اسکریپت پرسید، IP سرور خارج را وارد کنید.

این اسکریپت دقیقاً چه کاری انجام می‌دهد؟
شناسایی خودکار: خودش می‌گردد و Interface اصلی، IP پابلیک و مهم‌تر از همه Gateway MAC Address (که برای Paqet حیاتی است) را پیدا می‌کند.

تنظیمات امنیتی: دستورات پیچیده iptables (مثل NOTRACK و DROP RST) را که برای کارکرد صحیح Paqet لازم است، به‌طور خودکار اعمال و ذخیره می‌کند.

Forwarding: در سمت ایران، پورت 4789 (پورت استاندارد VXLAN/GRETAP) را فوروارد می‌کند و ip_forwarding را فعال می‌کند.

Service: یک سرویس سیستمی می‌سازد تا اگر سرور ریستارت شد، تانل خودکار بالا بیاید.
